UzTransformator WordPress Theme
===============================

Qanday o'rnatiladi:

1. Ushbu mavzuni ZIP ko'rinishida yuklab oling (ChatGPT'dan berilgan link).
2. WordPress admin paneliga kiring.
3. Appearance -> Themes -> Add New -> Upload Theme tugmasini bosing.
4. ZIP faylni tanlang va Yuklang (Install Now).
5. Aktivatsiya qiling (Activate).
6. Settings -> Reading -> Homepage Displays -> "A static page" -> Front Page sifatida yangi bo'sh sahifa tanlang *yoki* ushbu mavzu avtomatik front-page.php dan foydalanadi (ko'p hollarda yetarli).

Rasmlarni almashtirish:
- wp-content/themes/uztransformator/assets/images/ papkasiga o'z transformator rasmlaringizni yuklang.
- Fayl nomlari tmg-100.jpg, tmg-630.jpg va hokazo â€” o'zgartirmasangiz yaxshi.
- Yoki functions.php ichidagi product massivida o'zgartiring.

Narxlar va matnlarni o'zgartirish:
- functions.php -> uztransf_get_products() funksiyasida.
- Footer kontakt ma'lumotlari footer.php da.
